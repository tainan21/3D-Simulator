# Object Catalog

## Estruturas

- `fence`: parede base de `1.5m`
- `fence-tl`: parede alta com socket superior de `4m`
- `gate`: portal interativo, mesmo footprint, fluxo mutavel
- `tower`: defesa ancorada em `fence-tl`
- `base-core`: alvo principal de defesa

## Atores

- `player`: avatar base de `1m`
- `enemy`: hostil padrao de `2m`
- `dwarf`: hostil pequeno de `0.5m`
- `boss`: hostil estrutural de `4m`

## Conectores

- `ramp`
- `platform-link`
- `top-socket-link`
