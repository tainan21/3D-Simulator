# Ponto Zero

Ponto zero eh a fundacao de v1 reutilizavel para qualquer game deste universo.

Aqui ficam contratos pequenos, legiveis e reaproveitaveis:

- catalogo de objetos
- criterios de v1
- mapa de implementacao

O objetivo nao eh substituir o codigo principal. O objetivo eh congelar o nucleo para que expansao nao desalinhe a verdade do projeto.
